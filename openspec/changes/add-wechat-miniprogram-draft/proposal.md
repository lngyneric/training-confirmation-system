# Add WeChat Mini Program Support

## Summary
Add support for WeChat Mini Program to the Training Confirmation System, allowing users to view tasks and confirm them via WeChat.

## Motivation
- Increase accessibility for users who prefer WeChat.
- Enable push notifications and easier sharing within the WeChat ecosystem.

## Solution Overview
- Introduce a new `miniprogram` directory using **Taro** (React).
- Implement key features: Login, Task List, Task Details, Confirmation.
- Ensure the mini-program aligns with the existing web app's data structure.
